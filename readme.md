<h1 style="text-align: center;">Easyprint - Your conductor between printer and PC</h1>

<br><br>

<h1>Программа не кроссплатформенная и поддерживается только на windows</h1>
<h1>Программа умеет работать только со следуйщими типами докуменов: PDF, WORD, RTF, TXT, DOCX, ODT, </h1>

<br>

<h3><b>Программа прослушивает определённую папку и в случае появление новых файлов компьютер запускает печать</b></h3>
<hr>



<h2>
<b>Используемые теги в настройках:</b>
<ul>

<li> "TARGET" - название прослушиваемой папки
<li> "INTERVAL" - интервал проверки, можно изменить только вручную(по умолчанию 10)
<li> "DELETE_AFTER_PRINT" - boolean тег который определяет будет ли удаляться файл после печати(1: да, 0: нет)
<li> "IGNORE_EXISTING" - boolean тег который определяет будет ли программа игнорировать файлы которые уже были созданы на момент запуска программы(1: да, 0: нет)
<li> "DEFAULT_PRINTER" - string тег который хранит точное название принтера


</ul>
</h2>

<h2>
<b>Аргументы основной программы(EasyPrint.exe):</b>
<ul>

<li> --updated : Аргумент который говорит основной программе что файлы были обновлены, и чтобы завершить обновление надо обновить сам лаунчер. Лаунчер самостоятельно запускает программу с этим аргументом в случае успешного обновления.
<li> --manually_update : Аргумент позволяет вручную запустить процесс обновления пропуская проверку наличие новых обновлений. Можно использовать для восстановления файлов

</ul>
</h2>

<h2>
<b>Коды исключений: (std::runtime_error)</b>
<ul>

<li> 01 - Конфигурационный файл не найден. Ошибка возникает при вызове JSON_worker::LoadJSONTags, когда указанный файл не существует
<li> 02 - Не удалось создать конфигурационный файл. Обычно возникает, когда программе недостаточно прав для создания файла. Попробуйте запустить от имени администратора или переместить исполняемый файл в другую папку
<li> 03 - Не удалось прочитать конфигурационный файл. Возникает по неизвестным причинам
<li> 04 - Расширение файла пустое
<li> 05 - Конфигурация повреждена. Программа не смогла загрузить все параметры из файла. Если перезапуск не помогает — обратитесь к разработчику
<li> 17 - Расширение файла пустое
<li> 18 - Не удалось найти файл в целевой папке (targer folder)
<li> 19 - Программа запустилась для другого расширения файла
<li> 20 - Не удалось найти pandoc

<li> 21 - Не удалось конвертировать файл в pdf
<li> 22 - функция для печати специфичных файлов для этого расширения не определена

</ul>

<b>Launcher</b>
<ul>

<li> 11 - Не удалось проверить наличие обновлений: программа не смогла открыть Version.txt
<li> 12 - Не удалось проверить наличие обновлений: локальный Version.txt пустой
<li> 13 - Не удалось проверить наличие обновлений: нет подключения к интернету (HTTP status code != 200)
<li> 14 - Не удалось проверить наличие обновлений: Version.txt на GitHub пустой
<li> 15 - Не удалось проверить наличие обновлений: программе не удалось преобразовать строки в числа
<li> 16 - Не удалось создать файл для скачивания обновлений
</ul>
