<h1 style="text-align: center;">Easyprint - Your distance print helper</h1>

<br><br>

<h1>Программа не кроссплатформенная и поддерживается только на windows</h1>
<h1>Программа умеет работать только со следуйщими типами докуменов: PDF, WORD, RTF, TXT, DOCX, ODT, </h1>

<br>

<h3><b>Программа прослушивает определённую папку и в случае появление новых файлов компьютер запускает печать</b></h3>
<hr>



<h2>
<b>Используемые теги в настройках:</b>
<ul>

<li> "TARGET" - название прослушиваемой папки
<li> "INTERVAL" - интервал проверки, можно изменить только вручную(по умолчанию 10)
<li> "DELETE_AFTER_PRINT" - boolean тег который определяет будет ли удаляться файл после печати(1: да, 0: нет)
<li> "IGNORE_EXISTING" - boolean тег который определяет будет ли программа игнорировать файлы которые уже были созданы на момент запуска программы(1: да, 0: нет)
<li> "PRINT_SOUND_NOTIFICATION" - boolean тег который определяет будет ли программа воспроизводить звук при запуске печати(через a)(1: да, 0: нет) - В разработке. Частично готово

</ul>
</h2>

<h2>
<b>Коды исключений: (std::runtime_error)</b>
<ul>

<li> 01 - конфигурационный файл не найден. Ошибка возникает когда вызывает метод JSON_worker::LoadJSONTags с аргументом filename который не существует
<li> 02 - не удалось создать конфигурационный файл. Ошибка обычно возникает когда программе недостаточно прав чтобы создать файл. Попробуйте запустить программу от имени администратора или переместить исполняемый файл в другую папку
<li> 03 - не удаётся прочитать конфигурационный файл. Ошибка возникает по неизвестным причинам
<li> 04 - Не соответствие типов. Ошибка вызывается когда программа при определении расширения файла совершила ошибку и вызвала не тот метод для печати
<li> 05 - Конфигурация повреждена. Ошибка вылезает когда программа по какой то причине не смогла загрузить все параметры из файла. Если перезапуск программы не помогает обратитесь к разработчику
</ul>

<b>Launcher</b>
<ul>

<li> 11 - Не удалось проверить наличие обновления из за того что программа не смогла открыть Version.txt
<li> 12 - Не удалось проверить наличие обновления из за того что локальный Version.txt пустой
<li> 13 - Не удалось проверить наличие обновления из за отсутствие подключения к интернету (code != 200)
<li> 14 - Не удалось проверить наличие обновления из за того что Version.txt пустой(github)
<li> 15 - Не удалось проверить наличие обновления из за того что программе не удалось преобразовать строки в числа
<li> 16 - Не удалось создать файл на который будет осуществляться скачка

</ul>
